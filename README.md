# Smart Schematic V3 (Demo)

Smart Schematic V3 is an interactive, web-based schematic viewer designed to
demonstrate how traditional schematics can become assembly-aware,
clickable, and visually informative.

This demo focuses on usability, clarity, and manufacturing insight.
